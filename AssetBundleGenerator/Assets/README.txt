How to generate an ARLearn Asset Bundle:
1. Set up the prefabs in the AssetBundleGenerator scene.
2. Select them all and scale them so they fit in the 1x1x1 scaling cube.
3. Select all prefabs in the project view, right click, and press 'Create Asset Bundle'.

Your asset bundle should be located in 'Assets/Bundles/'.
You can now upload it into our website at https://www.arlearn.xyz
